## FAQ

### 電子版の販売はありますか？

電子版については、kindle版やkobo版などリフロー型を販売予定です。5月29日の配信を目指して、作業を進めているとのことです。

### Kaggle Notebookの「Commit」が存在しません

2020年4月2日ごろにKaggle Notebookのデザインが更新されました。新デザインでのCommit方法は、[動画](https://youtu.be/2kMzUfajj1U)で解説しています。
